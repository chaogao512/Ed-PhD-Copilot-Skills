# Qualitative Methods Minimum Reporting

| Method | Use when | Must report |
|---|---|---|
| Semi-structured interview | Understanding perception, mechanism and resistance | Sampling, protocol, coding process, representative quotes |
| Focus group | Exploring shared norms or stakeholder negotiation | Group composition, facilitation, interaction dynamics |
| Observation | Capturing real practice and workflow | Site, duration, observation focus, field-note protocol |
| Document analysis | Studying policy, rules or institutional traces | Corpus, coding scheme, inclusion criteria |
| Expert review | Evaluating framework or artifact | Expert criteria, review questions, revision decisions |

## Trustworthiness checks

Use at least two where feasible: member checking, peer debriefing, audit trail, double coding, negative case analysis, thick description.
