# Governance Figure Types

| Figure type | Use when | Core visual question |
|---|---|---|
| Macro governance framework | Explaining policy, organization and technology relationship | What levels and values guide the system? |
| Layered smart campus architecture | Presenting platform or Agent architecture | How do data, intelligence, applications and oversight connect? |
| Multi-actor collaboration diagram | Showing administrators, teachers, students, systems and experts | Who acts, decides, reviews and is accountable? |
| Human-AI feedback loop | Explaining decision support or evaluation | Where does human interpretation and override happen? |
| Indicator system tree | Presenting evaluation framework | How are dimensions and indicators organized and validated? |
| Evidence chain diagram | Explaining mixed-methods validation | How do multiple data sources support claims? |
