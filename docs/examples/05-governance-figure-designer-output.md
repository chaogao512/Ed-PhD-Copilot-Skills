## Figure type

Layered smart campus governance architecture + human-AI feedback loop.

## Recommended paradigm

采用“五层堆叠架构 + 右侧人类监督责任链”的组合图。

## Layout sketch

从下到上排列五层：

1. **基础设施与数据采集层**：教务系统、学工系统、LMS、教学评价、资源调度。
2. **数据治理中台层**：数据标准、权限控制、脱敏处理、质量检查、生命周期管理。
3. **多 Agent 协同层**：情境感知 Agent、风险预警 Agent、资源调度 Agent、教学质量 Agent。
4. **业务应用层**：学业支持、教学改进、学生服务、领导决策。
5. **治理反馈层**：行动执行、效果评估、规则修订、组织学习。

右侧设置纵向责任链：教师复核、管理者决策、专家审查、伦理监督、申诉反馈。每一层与责任链之间用虚线连接，表示人工解释、复核与问责。

## Arrow rules

- 实线箭头：数据流。
- 双向箭头：协同决策。
- 虚线箭头：人工复核与解释。
- 环形箭头：反馈与持续改进。
