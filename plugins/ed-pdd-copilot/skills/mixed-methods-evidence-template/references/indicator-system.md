# Indicator System Construction Guide

## Minimum workflow

1. Define the construct and theory anchor.
2. Extract candidate indicators from policy, literature and practice cases.
3. Code and merge indicators with transparent inclusion rules.
4. Conduct expert consultation or Delphi rounds.
5. Test reliability and validity when questionnaire data are collected.
6. Determine weights only after the indicator meaning is stable.
7. Report limitations and context of use.

## Delphi reporting essentials

- Expert selection criteria and number.
- Number of rounds.
- Consensus threshold.
- Revision rule between rounds.
- Authority/familiarity or equivalent expert credibility evidence when used.

## Weighting cautions

AHP, entropy weighting and combined weighting are tools, not proof of validity. Weighting cannot repair a poorly theorized indicator system.
