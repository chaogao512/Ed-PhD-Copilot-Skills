# Verification Checklist

Before using AI-assisted output, verify:

- Source traceability: every citation or policy exists.
- Claim-evidence match: claims are supported by the right evidence type.
- Method consistency: design, data and analysis align.
- Ethics: consent, privacy, fairness and human oversight are addressed.
- Voice: text reflects the user's actual research and does not add unsupported claims.
- Disclosure: AI use is disclosed when required by institution or venue.

## Chinese Literature Verification Gate

When AI helps with Chinese EdTech, CSSCI or core-journal literature:

| Item | Required check |
|---|---|
| Bibliographic completeness | Author, title, journal/source, year, volume/issue/pages and DOI/CNKI URL when available |
| Traceable source | CNKI detail page, journal website, DOI/publisher page, local PDF or school database export |
| Name disambiguation | Check same-name authors and variant names such as 顾小清 / 顾小青 |
| Status label | Mark each item as verified, candidate, local-plan reference or failed-to-verify |
| Citation boundary | Candidate records may guide search but must not appear in formal references |

If only a search snippet, repost, AI memory or incomplete title is available, keep the item in a candidate table and state that it is not citeable yet.

## AI Use Disclosure Checklist

Before submission, prepare a disclosure note when AI was used materially.

| Check | Required action |
|---|---|
| Tool identity | Record tool/model name when known and access date or version when relevant |
| Task scope | State whether AI assisted literature organization, coding, analysis code, translation, writing, figures or review |
| Human verification | State that the researcher verified sources, claims, data outputs and final text |
| No authorship | Do not list AI as author or contributor with responsibility |
| Venue alignment | Check target journal, university and funder AI policy |
| Sensitive data | Confirm no unauthorized personal, classroom, institutional or confidential peer-review data were uploaded |

### Example disclosure language

> AI tools were used to assist with outline organization, language revision and checklist-based consistency review. The author verified all citations, evidence, analysis decisions and final claims and remains responsible for the manuscript.

If AI assisted coding, data analysis or figure generation, replace the generic sentence with a task-specific statement.

For venue-specific wording, read `venue-disclosure-templates.md` and adapt the relevant template to the user's actual journal, university, ethics committee or funder.
