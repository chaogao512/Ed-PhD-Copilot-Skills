## Layout Sketch

```text
课堂活动与学习证据
        ↓ 数据最小化采集
AI 辅助分析：互动模式、作业表现、反馈建议
        ↓ 可解释建议
教师专业解释：情境判断、误差识别、反馈重写
        ↓
教研共同体讨论：同伴审议、改进策略
        ↓
学生反馈与申诉：理解评价、补充情境、纠错
        ↓
教学改进行动：教学设计调整、评价任务优化
        ↺ 回到课堂活动
监督边界：隐私保护、公平检查、用途限定、责任归属
```

右侧增加 `Model/System Card Panel`：

| Item | Content |
|---|---|
| Intended use | 辅助教师整理课堂评价证据和反馈建议 |
| Non-use boundary | 不得自动用于学生排名、惩戒、教师绩效或高风险管理决策 |
| Limitation | AI 可能误读情境、忽略学生差异或放大历史偏差 |
| Oversight | 教师复核、教研共同体审议、学生申诉纠错 |
| Monitoring | 误判记录、公平性检查、反馈采纳与修订日志 |

## Quality Self-Check

- 教师、教研团队和学生均在图中可见。
- 包含隐私、公平、用途限定和责任边界。
- 包含模型说明卡，明确适用场景、非用途边界、限制与监测。
