# Case: 区域中小学教师数字胜任力提升的组织支持机制研究

## Case Purpose

本案例用于测试 `Ed-PhD-Copilot-Skills` 是否能够处理教师发展与区域教育治理类主题。它区别于智能校园平台建设案例，重点考察技能能否把教师数字胜任力从“个人技能培训”转化为“组织支持、领导力、资源配置与持续改进机制”问题。

## Research Background

区域教育数字化转型要求教师能够合理使用数字资源、数据工具和智能技术改进教学、评价与专业发展。但现实中，教师数字能力提升常被简化为平台培训、工具打卡或证书考核，导致培训与课堂实践脱节、教师负担增加、学校支持不足、数字资源分布不均等问题。

## Core Governance Problem

如何构建一个由教育行政部门、学校管理者、教研组织、教师共同参与的组织支持机制，使教师数字胜任力提升从一次性培训转向持续性、情境化、证据驱动的专业发展过程。

## Stakeholders

- 区域教育行政部门：制定政策、配置资源、建立评价与支持机制。
- 学校管理者：提供校本支持、组织教研、协调时间和资源。
- 教研员与培训者：设计培训内容、提供课堂转化指导。
- 教师：在真实教学场景中使用数字技术并形成反思证据。
- 学生：作为教学改进的受益者，其学习体验和权益需要被保护。
- 平台与资源服务方：提供数据和资源，但不应替代教育判断。

## Initial Research Questions

1. 区域教师数字胜任力提升面临哪些组织性障碍？
2. 哪些组织支持因素能够促进教师将数字能力转化为教学改进行动？
3. 如何构建区域-学校-教研-教师四层协同的支持机制？
4. 该机制如何通过混合证据验证其可行性和改进效果？

## Candidate Theory Anchors

- 教师数字胜任力框架：用于界定能力结构和发展维度。
- 组织支持理论：用于解释资源、领导、同伴和制度支持如何影响教师行为。
- 活动理论：用于分析教师、工具、规则、共同体和分工之间的矛盾。
- 持续改进理论：用于设计诊断、反馈、改进和再评估循环。

## Evidence Route

- 政策与培训文件分析：识别区域目标和现有支持机制。
- 教师问卷：测量数字胜任力、组织支持感、数字疲劳和技术采纳。
- 半结构访谈：理解教师在课堂转化中的真实困难。
- 课堂与教研材料分析：检查培训成果是否进入教学设计和评价实践。
- 过程数据：记录培训参与、教研互动、资源使用和改进行动。

## Ethics and Data Governance

- 教师数据不得用于简单排名或惩罚。
- 学生学习数据只能作为课堂改进证据，需脱敏并避免个体标签化。
- 平台数据必须说明来源、授权、用途和保存周期。
- AI 生成的教师画像不得替代校本教研和人工判断。
